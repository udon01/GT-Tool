グランツーリスモ7の自動周回ツール
(擬似B-spec)

バックグラウンドでも普通に動かせた(キーボード・マウスの操作は入ってません)

↓のツールをインストールしてください
https://github.com/ViGEm/ViGEmBus/releases
(たぶん最新版でOK)
コントローラーが差してあると動かないので注意
合わせてPSリモートプレイを5.5.0にする
中にインストーラー入ってます
アプデをキャンセルすればそのまま使える(一応手順のとこに書いといた)

------------事前準備------------
チャンピオンシップの「アメリカ大陸選手権」を開放
(ゲームを進めると出ます)
「ダッジ SRT トマホーク S VGT」を購入してください(1億Cr.)
フロントタイヤはスポーツハード
リアタイヤはレーシングソフト
ギア比は自動設定380km位で
もし1億ない場合は設定の待ち時間を長くすれば別の車でもできます

キーコンフィグ
・ブーストを使う場合はR1にセット

アシストセッティング
・TCSを1以上に設定
・MTで遊んでいる方はATに

以下PC側の操作
「PSリモートプレイ」をダウンロードする
https://remoteplay.dl.playstation.net/remoteplay/lang/jp/ps5_win.html
ここからダウンロード
使えるようにする手順も載ってます

注意！
PC側にコントローラーがささってると動きません
線抜いといてね
--------------------------------


--------------手順--------------
①アプリを起動
②「この画面で開始」にカーソルを持っていく
③リモートプレイを起動、アプデをキャンセルして番号でPS5(PS4)とPCを接続
④アプリの「開始」をクリック
⑤停止するまで無限に繰り返します
--------------------------------


-------ロード時間について-------
PS5の場合は初期設定でOK
PS4の場合、
・エントリー後
・2戦目ロード
・2戦目棄権～メニュー
の3つを設定する必要があるみたいです

PS4 (HDD)
エントリー後 33秒
2戦目ロード 40秒
2戦目棄権～メニュー 15秒

PS4pro (SSD換装済み)
エントリー後 23秒
2戦目ロード 27秒
2戦目棄権～メニュー 11秒

※ほか色んな条件で変わると思うので目安程度に
--------------------------------


-------------注意点-------------
停止ボタンはレース終了後に動作します
すぐに終了させたい場合(接続が切れたなど)は右上の×ボタンを押してください。
